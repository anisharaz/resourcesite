# Learning Resource Management Application For Universities

### You can add subject-PDFs, question banks & importent Links in the data base and accordingly the webpage will load the content.<br>The PDFs docs can be directly view because the project is build using a inbult pdf rendered to render on web browser.<br><br>


The current structure of the application is.
> Branch Name
>> Subject List
>>> Modules(Pdf View)

The Subject module PDFs are placed in Public polder in the structure as <br> /public/{subject_code}/{subject_code}_{module_number}.pdf<br>
eg. /public/22CSE220/22CSE220_1.pdf
